atres::Renderer::HL_DEFINE_IS ( useIdeographWords  ,
UseIdeographWords   
)

When turned on, this regards ideographs as words when segmenting text into lines.

Note
This is particularly useful when having Japanese or Chinese text. Korean text has actual spaces so this is not necessary, but still useful.
